Types of css selectors
1) Element selector (h1{CSS})
2) Class selector (.className{CSS})
3) Id selector (#id{CSS})
4) Attribute selector (h1[attribute]{CSS} -- p[draggable]{CSS}> -- p[draggable="false"]{CSS})
5) Universal selector (*{CSS})
